# Cognisync v0.1.1

This release turns the reference implementation into a more complete operator loop.

## Highlights

- added `cognisync doctor` for workspace and adapter readiness checks
- added real ingest commands for local files, pdfs, urls, and local repositories
- added `cognisync compile` for the main scan, plan, execute, and lint loop
- added a second built-in frontier CLI preset for Gemini
- added a polished demo workspace flow and checked-in example garden

## Validation

- `python3 -m unittest discover -s tests -v`
- `/usr/bin/env PYTHONPYCACHEPREFIX=/tmp/cognisync-pyc python3 -m compileall src tests`
- `PYTHONPATH=src python3 -m cognisync doctor --workspace examples/research-garden --strict`
- `PYTHONPATH=src python3 -m cognisync lint --workspace examples/research-garden --strict`

## Notes

This is still a GitHub-first source release. PyPI publishing remains intentionally deferred.
