# Cognisync v0.1.2

This release deepens the ingest layer so raw artifacts arrive with more usable structure.

## Highlights

- `ingest url` now captures page metadata such as description, canonical URL, headings, discovered links, and content stats
- `ingest repo` now captures repository stats, language signals, recent commits, and a richer README-backed manifest
- operator docs now describe the richer ingest substrate more explicitly

## Validation

- `python3 -m unittest discover -s tests -v`
- `/usr/bin/env PYTHONPYCACHEPREFIX=/tmp/cognisync-pyc python3 -m compileall src tests`
- `PYTHONPATH=src python3 -m cognisync --help`
- `PYTHONPATH=src python3 -m cognisync doctor --workspace examples/research-garden --strict`
