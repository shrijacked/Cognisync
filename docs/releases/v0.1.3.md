# Cognisync v0.1.3

This release turns Cognisync into a more complete operator loop.

## Highlights

- added a built-in Claude Code preset and stronger adapter templating with `{prompt_text}`
- deepened ingest with PDF text sidecars, URL image capture, repository tree snapshots, and manifest-driven batch ingest
- enriched compile packets with raw artifact context so external agents see more useful substrate up front
- upgraded query reports with inline source ids and source blocks
- added `cognisync research` to run search, prompt generation, and optional answer filing as one command
- kept CI green by smoke-testing demo generation, workspace health checks, and the research surface

## Validation

- `python3 -m unittest discover -s tests -v`
- `/usr/bin/env PYTHONPYCACHEPREFIX=/tmp/cognisync-pyc python3 -m compileall src tests`
- `PYTHONPATH=src python3 -m cognisync --help`
- `PYTHONPATH=src python3 -m cognisync research --workspace examples/research-garden "knowledge garden loop"`
