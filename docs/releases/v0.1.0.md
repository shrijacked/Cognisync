# Cognisync v0.1.0

Initial public release of the Cognisync reference implementation.

## Highlights

- filesystem-first workspace scaffolding for `raw/`, `wiki/`, `outputs/`, and prompt packets
- deterministic scanning, indexing, compile planning, linting, and search over Markdown-first knowledge bases
- Markdown and Marp renderers for reusable reports and briefings
- built-in Codex CLI adapter preset with stdin prompt streaming and optional last-message capture
- GitHub Actions CI plus the initial contribution and community repo templates

## Validation

- `python3 -m unittest discover -s tests -v`
- `/usr/bin/env PYTHONPYCACHEPREFIX=/tmp/cognisync-pyc python3 -m compileall src tests`
- `PYTHONPATH=src python3 -m cognisync --help`

## Distribution

This release is GitHub-first. PyPI publishing is intentionally deferred until the
adapter and CLI contracts stabilize across a few public iterations.
