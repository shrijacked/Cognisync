# Research Garden Demo

This directory is a seeded Cognisync workspace that demonstrates the full filesystem
contract in a compact form.

## Start Here

- `raw/` contains the source notes
- `wiki/sources/` contains compiled summaries
- `wiki/concepts/` contains cross-source synthesis
- `wiki/queries/` shows how an answer can be filed back into the garden
- `outputs/` and `prompts/` show the generated artifacts
