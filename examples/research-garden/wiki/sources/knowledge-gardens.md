# Knowledge Gardens

This source frames the knowledge base itself as the product surface. Markdown files,
backlinks, and local assets make the system legible to both humans and LLMs.

## Highlights

- files are durable, inspectable, and versionable
- local-first tools like Obsidian stay useful because the wiki remains plain text
- new answers can be filed back into the same corpus
